La mémoire java

  • Rien à faire, c'est pas comme en C, pas besoin d'allouer de l'espace et de le libérer tout se fait tout seul avec le Garbage Collector
  • FAUX !!
  • Le GC ne peut nettoyer que ce que vous libérez !

La mémoire java

  • Chaque objet créé est d'abord retenu en mémoire par son créateur (parent)
  • Il est aussi retenu par les objets qui le référencent
  • Un objet retenu ne peut pas être nettoyé
